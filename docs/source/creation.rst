=====================
Jedgar creation notes
=====================

This describes how I built this project, mostly for reference.

Creation
--------
-  Make the project directory:
        `mkdir jedgar`
        `cd jedgar`
-  Mmake a venv:
        `python -m venv .venv`
- Starting the venv:
        `source .venv/bin/activate`
- Quitting the venv:
        `.venv/bin/deactivate`
